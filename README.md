# gopherclaw

I built a template repo that shows how you can leverage Golang (Go Programming Language to build proper AI Agents! This template repo fires up an AI Agent that uses LangChain, MCP, and Anthropic Claude as an LLM to work directly with your filesystem, including searching files, reading from files, writing to files, and listing directories/folders!

Leverages Go's concurrency, goroutines, and channels with AI Agents for parallel task execution via langchain, claude, pinecone, and mcps
